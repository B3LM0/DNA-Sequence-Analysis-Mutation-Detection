# Example FASTA Files

This directory contains example FASTA files for testing the bioinformatics application.

## Files:

1. **example1_simple.fasta**
   - Simple short DNA sequence
   - Good for basic testing

2. **example2_with_orf.fasta**
   - Contains Open Reading Frames (ORFs)
   - Has start codon (ATG) and stop codon (TAA)

3. **example3_reference.fasta** & **example4_variant.fasta**
   - Pair of sequences for comparison
   - Contains a single substitution mutation (Aâ†’C at position 15)

5. **example5_insulin.fasta**
   - Partial human insulin gene sequence
   - More realistic example with multiple ORFs

## Usage:

Copy and paste the content of these files into the application, or use the "Upload FASTA File" button.
